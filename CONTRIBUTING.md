# Contributing

Thanks for contributing to AvenEditor.

## Development Setup

1. Install dependencies: `npm install`
2. Start fullstack dev server: `npm run dev`
3. Run checks before opening PR: `npm run lint`

## Pull Request Guidelines

1. Keep PRs focused. Split unrelated changes into separate PRs.
2. Include a clear problem statement and a concise change summary.
3. Add or update docs when behavior or configuration changes.
4. Keep TypeScript checks passing.

## Commit Style

Use short, imperative commit messages, e.g.:

- `fix: avoid stale remote cache in readonly mode`
- `docs: update server mode environment variables`

## Reporting Bugs

When reporting issues, include:

1. OS and Node.js version
2. Reproduction steps
3. Expected vs actual behavior
4. Relevant logs or screenshots
