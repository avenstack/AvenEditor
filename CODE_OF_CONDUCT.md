# Code Of Conduct

## Our Standard

We expect all participants to be respectful, constructive, and inclusive.

Examples of expected behavior:

- Give and receive feedback professionally.
- Focus criticism on ideas, not people.
- Assume good intent and seek clarity early.

Unacceptable behavior includes harassment, discrimination, personal attacks, and any abusive conduct.

## Scope

This Code of Conduct applies to project spaces, including issues, pull requests, discussions, and related communication channels.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that violate this Code of Conduct.

For serious or repeated violations, maintainers may temporarily or permanently ban contributors from project participation.
